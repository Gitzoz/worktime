Help

This app stores your entries locally in this browser.

  • Use Export JSON to download a backup file you can keep in a local folder.
  • Use Export CSV when you want to open the saved entries in Excel, Numbers, LibreOffice, or Google Sheets.
  • Add a comment to a day when you want to mark vacation, sick leave, a public holiday, or a short work note.
  • Use the file picker to import a previous backup. You can merge it with current entries or replace everything.
  • A running live timer is stored locally, but export includes saved workday entries. Stop work before exporting today.
  • Before moving from a local file to GitHub Pages, another browser, or another device, export JSON first and import it there.
  • Clearing browser data, using private browsing, or switching browsers may remove local entries unless you exported a backup.
  • Some browsers also show optional buttons for opening or saving a chosen backup file directly.
